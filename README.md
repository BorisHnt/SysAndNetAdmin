# SysAndNetAdmin

Site statique local de révision pour les sujets système et réseau du Common Core.

## Utilisation

Ouvrir `index.html` dans un navigateur.

Le site ne copie pas les sujets originaux. Il sert de notebook de révision avec notions, commandes utiles, erreurs fréquentes et réflexes de défense.

## Structure

- `index.html` : page principale
- `styles/main.css` : mise en page
- `scripts/app.js` : navigation et recherche
- `data/topics.js` : contenu de révision
