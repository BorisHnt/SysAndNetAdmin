# SysAndNetAdmin

Site statique local de révision pour les sujets système et réseau du Common Core :

- `05 - Born2beRoot`
- `06 - Net_Practice`
- `07 - Inception`
- `net_practice.1.9`

## Utilisation

Ouvrir `index.html` dans un navigateur.

Le site ne copie pas les sujets originaux et ne contient pas les fichiers de l’outil `net_practice`. Il sert de notebook de révision avec notions, commandes utiles, erreurs fréquentes et réflexes de défense.

## Structure

- `index.html` : page principale
- `styles/main.css` : mise en page
- `scripts/app.js` : navigation et recherche
- `data/topics.js` : contenu de révision
